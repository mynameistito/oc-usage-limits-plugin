Add Synthetic and MiniMax Token Plan providers. Both surface 5-hour rolling and weekly quota windows in the sidebar. Credentials are looked up in this order: (1) the configured `authPath` JSON file, (2) OpenCode's shared `auth.json`, then (3) the provider's `apiKey` config (with `{env:...}` references). MiniMax also activates the prompt footer for sessions using the `minimax-coding-plan` OpenCode provider.
